\begin{DoxyEnumerate}
\item Create a build directory\+: {\ttfamily mkdir build/}
\item Change the directory path (absolute path) to {\ttfamily libtorch} in C\+Make\+Lists.\+txt file. Then configure using C\+Make\+: {\ttfamily cmake ..}
\item Create an output directory (in the build directory) to store the results \& save checkpoints\+: {\ttfamily mkdir output/}
\item Build your project\+: {\ttfamily make}
\item Execute\+: {\ttfamily ./bin/example/}
\item The saved checkpoints and outputs will be at {\ttfamily output/} directory
\item To visualize, go back to the main directory\+: {\ttfamily cd ../}
\item Execute\+: {\ttfamily python3 utils/display\+\_\+samples.\+py}
\item The outputs will be stores in {\ttfamily build/output/output\+\_\+images/} directory
\item If you want to make an animation, run\+: {\ttfamily python3 utils/visualize.\+py} and it will save the animation for you in {\ttfamily build/output/output\+\_\+animation/} directory
